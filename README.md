# 🧠 AI Stress Detector

## 📌 Project Description
This project uses Hugging Face Transformers to analyze user emotions and predict stress levels based on text and lifestyle inputs.

## 🚀 Features
- Text-based stress detection
- Lifestyle analysis (sleep + study)
- Real-time predictions using Gradio

## 🛠️ Tech Stack
- Python
- Hugging Face Transformers
- Gradio

## ▶️ Run Locally
```bash
pip install -r requirements.txt
python app.py
